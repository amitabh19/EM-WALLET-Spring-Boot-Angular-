EM Wallet

A simple Wallet application made using Spring and Angular Framework

The application provides several features.
Fast Deposit and Fast withdrawal.
Fast money transer.
History of all transactions.
Unique Account Number, Email Id and Mobile Number for each user.
To prevent duplicity.

Modules Created

Deposit Anytime

This Module will help in Fast Deposit to the user's account

Withdraw Anytime

This Module will help in Fast withdrawal from the user's account

Send Money

This Module will help in transfer money to another user of the wallet

Transaction History

This Module will help to show history of all transactions performed by the user.
